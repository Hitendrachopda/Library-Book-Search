Very Simple, search books from the library by Book name, Author, Publisher, Price or Rating. 


== Description ==

The "Libraty Book Search" will be helpful for search the books by its Name or price or rating etc.

Plugin will simply generate shortcode and we can use it wherever we want to display Book search Function.

== Installation ==

1. Upload `library-book-search` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the *Plugins* menu in WordPress.
3. Add the shortcode to a post or page or use in custom templates.

= Usage =

	-> You can use this Shortcode `[library-book-search]`
	
== Frequently Asked Questions ==
